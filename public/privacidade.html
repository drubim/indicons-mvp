Política de Privacidade

1. Dados coletados

Coletamos dados fornecidos voluntariamente, como nome, telefone, e-mail e informações de interesse.

2. Finalidade

Os dados são utilizados para contato, atendimento, gestão de indicações e cumprimento legal.

3. Compartilhamento

Os dados poderão ser compartilhados com parceiros e administradoras exclusivamente para a finalidade solicitada.

4. Segurança

Adotamos medidas técnicas e administrativas para proteger os dados.

5. Direitos do titular

O titular pode solicitar acesso, correção ou exclusão dos dados conforme LGPD.

6. Consentimento

Ao utilizar a plataforma, o usuário consente com esta Política.
